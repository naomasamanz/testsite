# 🚀 テストログインフォーム（Web制作練習用）

これはWeb制作の練習で作っているテストログインフォームです！
土台になるので、どんどんフォークして、自分なりに変えてみたり、何かと連携してみたりしてみてね！

### 🌟 この土台でできること
* **JavaScriptでパスワードを判定**
  - 自分の好きなパスワードに変えてみてね！
* **ログイン後の画面（success.html）**
  - 自分の好きなデザインに変えてみてね！
* **エラー画面完備**
  - 404（迷子）や 403（立入禁止）も入っているよ！
